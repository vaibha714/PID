# 🚗 Cruise Control PID Simulation

Models and graphs the impact of a PID controller on a simulated cruise control environment. Users can tune PID values using sliders on a tkinter GUI. 


## ✨ Features
* Real-time graphical updates
* Adjustable Kp, Ki, Kd, setpointm and intital velocity values 
* Customizable drag and mass parameters for car
